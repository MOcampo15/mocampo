As a software engineer, there are plenty of career choices as there are different branches and ways in which software engineering can be used.

Careers:
Title Description
Applications Development Use problem-solving skills to create software that is non-web based
Systems Development Creation of coding background software as well as the design
Web Development Work on the creation of software that will run on the web and use HTML, PHP and JavaScript
Embedded Systems Development Work on the creation of software as well as computing systems on devices such as cars, will use assembly-languages and C

Skills:

  • Bachelor's degree in Software Engineering or Information Technology
  • Preferably 7+ years with the development of software and/or experience with programming
  • Good understanding of HTML5, CSS3, and CMS's
  • Problem-solving skills
  • Ability to work well in a team
  • Ability to communicate well both written and verbally
